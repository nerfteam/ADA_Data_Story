Sentiment Analysis

it's all about emotion...

It is common for people to talk about the disastrous effects of climate change as well as hope for humanity. For this reason it might be interesting to study the patterns of sentiment in what people say. The sentiment of a quote reveals very little about someones opinions on climate change and the message they convey. Additionally, it can let us inquire into the overall atmosphere of one's statements.

To determine the sentiment we can use a pretrained model from Huggingface. After running this model we append a value between -1 and 1 to each quote - where negative values imply a negative sentiment and vice versa. This model achieves state-of-the-art accuracy of 92.7%.

Let us inspect the positive and negative sentiment of climate change quotes in respect to political parties.

Based on the graphical representation above, we can see that the Democratic Party has a tendecy to cite its climate change quotes mostly with positive sentiment.

Furthermore, Democrats have more quotes with negative sentiment, and also more quotes in total. However, if we consider the ratio (i.e. negative_climate_change_quotes / total_num) we can conclude that Republicans (ratio = 0.55) are more likely to cite their qotes with negative sentiment than the Democrats (ratio = 0.43).

Moreover, we will analyse misinformations related with climate change quotes.
The key feature for this process has been extracted using CARDS (Computer-assisted detection and classification of misinformation about climate change) model. This feature takes integer values from the range [0,17]. The value 0 refers to quote not being classified as misinformation, while integer values from rage [1-17] refer to different classes of misinformations. For our analysis, we will focus only on the question if misinformation exists, and, therefore, we will consider only 2 classes: true quote (i.e. value of 0) and misinformation (i.e. any integer value from range [1-17].
So let's start.
